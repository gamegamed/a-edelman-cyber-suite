# a-edelman-cyber-suite
A retro arcade gaming diagnostic suite for linux
Ever wondered why your gaming PC isn't performing well? I created this tool just for you! This suite offers 10 different options to diagnose performance and network problems. Simply type a number, and it will instantly tell you if your system has that issue.
## How to Run
Open your Linux terminal and run the following commands:
```bash
wget [https://raw.githubusercontent.com/gamegamed/a-edelman-cyber-suite/main/nexus_v10.sh](https://raw.githubusercontent.com/gamegamed/a-edelman-cyber-suite/main/nexus_v10.sh)
chmod +x nexus_v10.sh
./nexus_v10.sh
